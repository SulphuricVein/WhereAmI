# 📍 WhereAmI

A simple Android app that tells you **where you are** — instantly.

Enter or detect your location, and the app displays the **area name** (like Jayanagar, a park, or nearby locality) in a clean and minimal interface.

---

## 🚀 Features

- 📍 Detects current location  
- 🏷️ Displays area/locality name  
- ⚡ Fast and lightweight  
- 🧩 Simple, minimal UI  

---

## 🧪 Current Status

**v0.1-alpha**

This is an early prototype focused on core functionality.
